This is an example of how to integrate the Garda Racing Chat Widget into any website.

Integration Steps:

  1. Include React and ReactDOM (if not already included)
  2. Include the widget CSS and JS files
  3. Configure and initialize the widget

Required Scripts:

<!-- React (if not already included) --> <script crossorigin src="https://unpkg.com/react@18/umd/react.production.min.js"></script> <script crossorigin src="https://unpkg.com/react-dom@18/umd/react-dom.production.min.js"></script> <!-- Garda Racing Chat Widget --> <link rel="stylesheet" href="./dist/style.css"> <script src="./dist/garda-chat-widget.umd.js"></script>

Configuration:

<script> // Configure the widget before it loads window.gardaChatWidgetConfig = { position: 'bottom-right', // or 'bottom-left' theme: 'light', // or 'dark' primaryColor: '#0066cc', supabaseUrl: 'YOUR_SUPABASE_URL', supabaseAnonKey: 'YOUR_SUPABASE_ANON_KEY' }; // Or initialize manually after page load document.addEventListener('DOMContentLoaded', function() { window.GardaRacingChatWidget.init({ position: 'bottom-right', theme: 'light', primaryColor: '#0066cc' }); }); </script>

The chat widget should appear in the bottom-right corner of this page!
